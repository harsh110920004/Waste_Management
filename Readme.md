## 🧪 How It Works

1. User uploads an image 📷  
2. Image is uploaded to Cloudinary ☁️  
3. AI analyzes image 🤖  
4. System classifies:
   - ✅ Waste  
   - ❌ Not Waste  
5. Data stored in MongoDB 🗄️  
6. Displayed in dashboard 📊
